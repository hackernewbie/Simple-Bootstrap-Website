This is a header for a fixed-width layout

To use a full-width layout use 'container-fluid' as the class of the container div.

Twitter Bootstrap divides the entire width of the website into 12 columns

It also uses 4 different classes for different types of (sizes) of screens. They are:

  • xs - Extra-small: For phones
  • sm - Small: For Tablets
  • md - Medium: For Standard Monitors
  • lg - Large: For ultra-large displays like iMacs and stuff
